# Sudoku-Solver
A Simple Sudoku Solver.

This is designed to solve sudoku puzzles.

It can solve 9x9, 12x12, 15x15 puzzles.

For complex puzzles, it may take long time.

Serial number: 08, Date of creation: 10 February 2022.
